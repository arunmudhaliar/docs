#include <qstring.h>

A class for flexible string handling using UT_string.

Manages strings and provides various constructors, methods for string manipulation, and overloads for common operations. Utilizes the UT_string library for internal management.

class timer
#include <timer.h>

Manages the timing and frame rate for the game.

This class provides functions to initialize, update, and reset the timer, as well as to retrieve various time-related values such as delta time, frame rate, and elapsed time.
